## 한국 시장 특이점 체크리스트

스캔 시 다음 한국 시장 특성을 반영하세요:

- [ ] **카카오 생태계**: 카카오톡 기반 서비스 연동 가능성
- [ ] **네이버 생태계**: 스마트스토어, 검색 기반 서비스 가능성
- [ ] **공구(공동구매) 문화**: 한국 특화 SNS 공동구매 시장
- [ ] **배달 인프라**: 세계 최고 수준 배달/물류 인프라 활용
- [ ] **고령화 시장**: 5060 디지털 소비자 증가
- [ ] **K-콘텐츠 수출**: K-뷰티, K-푸드, K-팝 해외 수요
- [ ] **교육열**: 에듀테크 시장 특수성
- [ ] **부동산 민감도**: 프롭테크 기회
- [ ] **의료 시스템**: 원격 진료 규제 환경
- [ ] **핀테크 규제**: 금융 규제 특수성
