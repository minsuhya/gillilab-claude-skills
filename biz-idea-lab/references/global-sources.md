## 검색 쿼리 템플릿

### 트렌드 스캔
```
"breakout startup [YEAR]"
"hot startup [CATEGORY] [YEAR]"
"[CATEGORY] market growing [YEAR]"
"AI [CATEGORY] startup funding [YEAR]"
```

### Pain Point 발굴
```
"wish there was an app for"
"frustrated with [TOOL] because"
"is there a tool that"
"[CATEGORY] is broken"
"[TOOL] sucks because"
```

### 시장 갭 발굴
```
"[CATEGORY] has no good solution"
"nobody has solved [PROBLEM]"
"[CATEGORY] needs disruption"
"why is [PROCESS] still manual"
```

### 성공 사례 학습
```
"[COMPANY] acquisition [YEAR]"
"[COMPANY] series A [YEAR]"
"[CATEGORY] unicorn [YEAR]"
"[COUNTRY] version of [GLOBAL_PRODUCT]"
```
